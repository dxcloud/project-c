Detailed Description

template<class C>
class sac2::GraphicAsset< C >

Derived from class Asset, provides methods for any graphic manipulation.

This is also template class, and encapsulates graphic resource. Same for Asset, this class cannot be instanciated, or copied directly.

See also:
Asset, AssetManager
