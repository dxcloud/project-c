Main namespace.

Configure from file VideoMode

  • width
  • height
  • bitsPerPixel ContextSettings:
  • depthBits (DB)
  • stencilBits (SB)
  • antialiasingLevel (AAL)
  • majorVersion (MaV)
  • minorVersion (MiV) RenderWindow
  • setVerticalSyncEnabled (vSync)
  • setMouseCursorVisible (mouse)
  • setKeyRepeatEnabled (keyR)
  • setFramerateLimit (FRL)
