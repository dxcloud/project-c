Tools

This page describes the use of SRG (SaC2 Resource Generator).

Resource Generator

The program srg is able to generate both C++ source and header files from XML. The generates files are required in order to use correctly sac2::AssetManager.

The program is written by using an extern library rapidxml to parse XML.

Compilation and Installation

The directory SaC2/tools/srg should provide Makefile which the command make may be used to compile and install this program.

The default installation path is SaC2/tools/bin. Use also the script sac2.sh at the root directory of SaC2 to set environment variables.

Usage

With all SaC2 environment variables defined, the following command invokes srg program. The argument FILENAME is required. The default generated filenames are resource.hpp and resource.cpp 

 $ srg FILENAME

XML structure

The XML file must have 3 levels of hierarchy.

  • Type of XML file, in this case: <resource> and </resource>.
  • Type of resource: image, sound, font, etc.
  • Resource items

Example

 <resource dir="samples">
   <image dir="images">
     <item id="IMAGE1" filename="image1.png" />
   </image>
 </resource>

Elements may be included (options):
  • Specify output filename (non extension): '--output=FILE'
  • Verbose: '--verbose', '-v'
  • Source/Header file suffix: '--src-suffix=SUFF', '--hdr-suffix=SUFF'
  • Help: '--help', '-h'
  • Version: '--version', '-V'
  • Warning: '--warning', '-w'
  • Disable documentation comment: '--no-comment', '-n'
  • Preserve order: '--same-order', '-s'
  • To stdout: '--to-stdout', '-O'
  • Ignore resource: '--ignore-image', '--ignore-sound', etc.
  • Add files: '--add-file=FILE'

Valid HDR suffixes: hh, H, hpp (default), hxx, HPP, h++ Valid SRC suffixes: cc, C, cpp (default), cxx, CPP, c++
