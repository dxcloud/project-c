Detailed Description

Class provides asset management utilities.

This class is a central class for managing assets. Resource may include Texture, Font, etc, and are loaded from files. The AssetManager contain a list of all asset filenames. This list must be initialized by the user, and cannot be modified in runtime.

Note:
This class is implemented with Singleton Pattern

status_t sac2::AssetManager::use_texture ( asset_id_t  id,
SpriteAsset &  sprite 
)

Load the sprite by the texture specified by id.

This method will search the loaded texture list first. If the texture is NOT loaded yet, it will try to load the texture specified by id, and then load the sprite by the texture.

Parameters:
[in]idIdentifier of the texture to load.
[out]spriteSprite to load to. This paramter left inchanged whether the texture is NOT found.
Returns:
Status code.
Return values:
STATUS_SUCCESSSuccessfully set the texture.
STATUS_ERRORFailed to load the texture.
Note:
Check the returned status code before using the sprite.
